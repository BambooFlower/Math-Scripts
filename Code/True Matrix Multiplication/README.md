# Matrix multiplication shortcut
Lazy man's shortcut for Matrix multiplication. Observe that 

\begin{pmatrix} 3 & 3  \\ 3 & 3 \end{pmatrix}\begin{pmatrix}6 & 6 \\ 6 & 6 \end{pmatrix}=\begin{pmatrix} 36 & 36 \\ 36 & 36 \end{pmatrix}
